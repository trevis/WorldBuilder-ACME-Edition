# WorldBuilder - ACME Edition

A landscape and world-building tool for Asheron's Call.

Originally created by the [Chorizite](https://github.com/Chorizite) team. This is an independently maintained fork by Vanquish.

## Building

Requires .NET 8.0 SDK or later.

```bash
dotnet build WorldBuilder.slnx
```

## Running

```bash
dotnet run --project WorldBuilder.Desktop
```
